1.Difference between var, let, and const
var: Variables are created with var, but this is an old method and can sometimes cause problems.
let: After creating a variable, its value can be changed.
const: Once a variable is created, its value cannot be changed.

2.What is the Spread Operator (...)?
The spread operator can be used to copy or move the values ​​of an array or object to another location.

3.Difference between map(), filter(), and forEach()
map(): map() creates a new array by changing each element.
filter(): filter() creates a new array by keeping some elements according to specified conditions.
forEach(): forEach() only works on each element but does not create a new array.

4.What is an Arrow Function?
Arrow function is a simple, concise and modern way of writing functions.

5.What are Template Literals?
Using template literals, you can easily add variables or expressions to strings.
